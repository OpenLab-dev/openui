# Popover

Muestra contenido enriquecido en un portal, activado por un botón.

## Usar

```jsx
import * as React from 'react'
import {
  Popover,
  PopoverTrigger,
  PopoverContent,
  Button,
} from '@openui-org/react/components'

function PopoverExample() {
  return (
    <Popover>
      <PopoverTrigger asChild className="text-black-light">
        <Button>Abrir popover</Button>
      </PopoverTrigger>
      <PopoverContent>
        Coloque aquí el contenido de la ventana emergente.
      </PopoverContent>
    </Popover>
  )
}
```
