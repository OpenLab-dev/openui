# Installation

Requirements:

- [React 18](https://reactjs.org/) or later
- [Tailwind CSS 3](https://tailwindcss.com/) or later

Inside your React project directory, run the following:

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm add @openui-org/react
    ```
  </Tab>
  <Tab>
    ```bash
    npm i @openui-org/react
    ```
  </Tab>
  <Tab>
    ```bash
    yarn add @openui-org/react
    ```
  </Tab>
</Tabs>


## Tailwind CSS Setup

Open UI is built on top of Tailwind CSS, so you need to install Tailwind CSS first. You can follow the official installation guide to install Tailwind CSS. Then you need to add the following code to your `tailwind.config.js` file:

```jsx
// tailwind.config.js
import { react } from '@openui-org/react'

/** @type {import('tailwindcss').Config} */
export default {
  content: [
    //...
    './node_modules/@openui-org/react/dist/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [react],
}

```

## Example

Now, you can use the component you installed in your application:

<br/>

<Button>Press me</Button>

```jsx
import * as React from "react";
import { Button } from "@openui-org/react/components";

function App() {
  return (
    <Button>Press me</Button>
  );
}
```
