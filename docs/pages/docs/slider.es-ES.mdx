# Slider

Entrada en la que el usuario selecciona un valor dentro de un rango determinado.

## Usar

```jsx
import * as React from 'react';
import { Slider } from '@keystone-ui/core';

export function SliderExample() {
    return (
        <Slider defaultValue={[50]} max={100} step={1} />
    );
}
```
