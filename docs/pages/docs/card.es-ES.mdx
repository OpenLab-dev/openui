# Carta

Muestra una tarjeta con cabecera, contenido y pie de página.

## Usar

```jsx
import * as React from 'react'
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
  Button,
} from '@openui-org/react/components'

function App() {
  return (
    <Card className="w-96">
      <CardHeader>
        <div className="w-full h-72 bg-muted/80 rounded-sm"></div>
      </CardHeader>
      <CardContent>
        <div className="text-xl font-medium pb-4 flex justify-between w-full">
          <CardTitle className="w-fill">Titulo</CardTitle>
          <span>$95.00</span>
        </div>
        <span className="text-sm font-extralight">
          Lorem ipsum dolor sit amet consectetur adipisicing elit. Explicabo,
          voluptatum?
        </span>
      </CardContent>
      <CardFooter className="flex justify-between">
        <Button className="w-full">Añadir al Carrito</Button>
      </CardFooter>
    </Card>
  )
}
```
