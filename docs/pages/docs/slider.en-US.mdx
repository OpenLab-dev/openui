# Slider

An input where the user selects a value from within a given range.

## Usage

```jsx
import * as React from 'react';
import { Slider } from '@keystone-ui/core';

export function SliderExample() {
    return (
        <Slider defaultValue={[50]} max={100} step={1} />
    );
}
```
