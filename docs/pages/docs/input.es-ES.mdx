# Input

Muestra un campo de entrada de formulario o un componente con aspecto de campo de entrada.

## Usar

```jsx
import * as React from 'react'
import { Input } from '@openui-org/react'

function App() {
  return <Input placeholder="Escribe algo..." />
}
```
