# Comienza

## Instalación [#instalación]


Requisitos:

- [React 18](https://reactjs.org/) o posterior
- [Tailwind CSS 3](https://tailwindcss.com/) o posterior

Inside your React project directory, run the following:

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm add @openui-org/react
    ```
  </Tab>
  <Tab>
    ```bash
    npm i @openui-org/react
    ```
  </Tab>
  <Tab>
    ```bash
    yarn add @openui-org/react
    ```
  </Tab>
</Tabs>
