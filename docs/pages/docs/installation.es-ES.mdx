# Comienza

## Instalación

Requisitos:

- [React 18](https://reactjs.org/) o posterior
- [Tailwind CSS 3](https://tailwindcss.com/) o posterior

## Instalar

Dentro del directorio de su proyecto React, ejecute lo siguiente:

```sh npm2yarn
npm i @openui-org/react
```

<Steps />

## Tailwind CSS Configuración

Open UI está construida sobre Tailwind CSS, por lo que primero debe instalar Tailwind CSS. Puede seguir la guía de instalación oficial para instalar Tailwind CSS. Luego debe agregar el siguiente código a su archivo `tailwind.config.js`:

```jsx
// tailwind.config.js
import { react } from '@openui-org/react'

/** @type {import('tailwindcss').Config} */
export default {
  content: [
    //...
    './node_modules/@openui-org/react/dist/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [react],
}
```

## Example

Ahora puede utilizar el componente que instaló en su aplicación:

<br />

<Button>Press me</Button>

```jsx
import * as React from 'react'
import { Button } from '@openui-org/react/components'

function App() {
  return <Button>Press me</Button>
}
```
