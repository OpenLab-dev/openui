# Acordeón

Un conjunto apilado verticalmente de títulos interactivos que revelan cada uno una sección de contenido.

## Usar

```jsx
import * as React from "react";
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from '@openui-org/react/components'

function App() {
  return (  
    <Accordion type="multiple">
        <AccordionItem value="example">
            <AccordionTrigger>
                ¿Usar Acordeón?
            </AccordionTrigger>
            <AccordionContent>
                Lorem ipsum dolor sit amet consectetur adipisicing elit. Explicabo, voluptatum?
            </AccordionContent>
        </AccordionItem>
    </Accordion>
  );
}
```
